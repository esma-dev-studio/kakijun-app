# かきじゅんマスター 🖌️

ひらがな・カタカナ・小学1〜2年生の漢字、ぜんぶで **332文字** の書き順を楽しく覚えられる子ども向けウェブアプリです。

## あそびかた

1. ホームで「ひらがな / カタカナ / かんじ1ねんせい / かんじ2ねんせい」からえらぶ
2. 文字をタップする
3. **👀 みる** — 書き順アニメーションを再生(はやさ3だんかい・1画ずつ送り・番号表示)
4. **✏️ なぞる** — 赤い丸から指やマウスでなぞって練習。ぜんぶ書けたら花丸💮と星(★1〜3)がもらえる
5. 練習した文字には一覧で ⭐ がつき、ホームに進捗が表示される

## しくみ

- HTML / CSS / Vanilla JavaScript のみ(フレームワーク・ビルド不要)
- 書き順データは [KanjiVG](https://kanjivg.tagaini.net) の SVG から `tools/build_data.py` で抽出
- なぞり判定は「進捗ウィンドウ方式」: 自己交差する画でも誤判定せず、逆向き書きは方向一致度チェックで弾く(全1,930画で自動検証済み)
- 効果音は WebAudio 合成(音声ファイルなし)、進捗は localStorage に保存
- スマホのタッチ操作に最適化(PCマウスも対応)

## ファイル構成

```
index.html      画面構造
style.css       デザイン
js/app.js       画面遷移・進捗管理
js/player.js    書き順アニメーションエンジン
js/sounds.js    効果音 (WebAudio)
js/tracer.js    なぞり判定エンジン
data/*.js       書き順データ (KanjiVG 由来・自動生成)
tools/build_data.py  データ生成スクリプト
```
